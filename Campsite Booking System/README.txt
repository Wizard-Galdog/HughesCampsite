THIS APPLICATION WON'T FULLY FUNCTION WITHOUT THE SQL SERVER CONNECTION.

To make this application run on your pc:
You must have Visual Studio and Microsoft SQL Server Management installed
You must import the server to the ide, Use the HughesDatabase file that is in this folder(In same directory as this readme)
Then boot up Hughes Campsite on Visual Studio
Go to the the CollectionClass.cs file
Use Ctrl+F and search for "connectionString"
Change this to your connection String
Now you can run the final build executable.

Otherwise, The code of the program is still fully examinable in Visual Studio. 
Queries for the Sql tables can also be found.
